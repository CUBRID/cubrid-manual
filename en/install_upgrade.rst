************************
Installing and Upgrading
************************

This chapter contains useful information on installation, setup, upgrading and database migration. 

This chapter covers the following topics:

*   Installing 
*   Environment setup
*   Upgrading
*   Database Migration

For information about ports used by CUBRID for Windows, please refer :ref:`Configuring Environment <Configuring-Environment-on-Windows>` section; for Linux, every configuration except for **APPL_SERVER_PORT** is same for Windows.

You can download various CUBRID tools and drivers in `http://www.cubrid.org/downloads <http://www.cubrid.org/downloads>`_\ .
