******************
설치 및 업그레이드
******************

CUBRID가 사용하는 포트에 대해서는 :ref:`connect-to-cubrid-server`\ 을 참고한다.  Linux 환경에서는 **APPL_SERVER_PORT** 를 제외한 나머지 포트 설정이 Windows 환경에서의 포트 설정과 동일하다.

CUBRID의 다양한 도구 및 드라이버는 http://www.cubrid.org/downloads\ 에서 다운로드할 수 있다.
